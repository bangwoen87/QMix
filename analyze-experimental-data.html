5. Example #3: Analyzing experimental dataÂ¶

This notebook will show a brief example of how QMix can be used to analyze experimental data. Note that a wide range of parameters can be passed to these functions in order to import data that has been stored in different formats. Please see params in qmix/exp/parameters.py for a full list.

For the purposes of this notebook, example data has been included in a directory called eg-230-data/. This is experimental data that was taken from an SIS device illuminated by a local-oscillator (LO) at 230 GHz.

[1]:

%matplotlib inline

import qmix
import numpy as np
import matplotlib.pyplot as plt

There are two main classes that can be used to analyze experimental data:

  • qmix.exp.RawData0, which is used for DC data (no LO pumping), and
  • qmix.exp.RawData, which is used to analyze pumped data (with LO illumination).

5.1. Import DC I-V dataÂ¶

The DC experimental data is located in two different files:

  • dciv.csv: DC current vs. bias voltage (no LO pumping)
  • dcif.csv: IF power vs. bias voltage (no LO pumping)

Both files are CSV files with one column for voltage and one column for current or IF power.

We will now use the RawData0 class to load, filter, and analyze the DC data. It will also automatically correct any offsets in the current or voltage data, and calculate the IF noise based on the DC shot noise.

Note that by passing additional keyword arguments to RawData0, you can control how the CSV files are imported (i.e., changing which columns are imported, how large the header is, and what kind of delimited is used). Please see the documentation for RawData0 for more information.

5.2. Import Data at 230 GHzÂ¶

We will now use the RawData class to import I-V and IF data from the device when it is illuminated by an LO at 230 GHz.

The experimental files are:

  • f230.2_iv.csv: the pumped I-V curve
  • f230.2_if-hot.csv: the IF power measured with a hot blackbody load
  • f230.2_if-cold.csv: the IF power measured with a cold blackbody load

This class will automatically load, filter and analyze the data. This includes calculating the noise temperature and the properties of the embedding circuit.

[8]:

pump = qmix.exp.RawData('eg-230-data/f230.2_iv.csv',
                        dciv,
                        if_hot_file='eg-230-data/f230.2_if-hot.csv',
                        if_cold_file='eg-230-data/f230.2_if-cold.csv')

5.2.1. Plot embedding impedanceÂ¶

The RawData class also recovers the embedding circuit using the technique described by Skalare (1989) and Withington et al. (1995).

[11]:

# Plot the error surface

fig, ax = plt.subplots(figsize=(7,5))
pump.plot_error_surface(ax=ax);

This plot was calculated using the error function from Withington et al. (1995).

The minimum error represents the best impedance estimation.

[12]:

# Plot a simulated I-V curve using the recovered impedance

fig, ax = plt.subplots(figsize=(7,5))
pump.plot_simulated(ax=ax);
